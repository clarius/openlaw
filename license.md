Software Libre y Licencia Comercial de OpenLaw

Fundación Clarius ofrece OpenLaw bajo un modelo de doble licencia, 
adaptado a las necesidades de distribuidores comerciales y proyectos 
de código abierto.  

Para OEMs, ISVs, VARs y otros distribuidores de aplicaciones comerciales:  
Si combinás y distribuís software comercial con OpenLaw sin querer 
compartir el código bajo la AGPLv3, tenés que obtener una licencia 
comercial de [Fundación Clarius](mailto:ventas@clarius.org).

Para proyectos open source y desarrolladores de aplicaciones libres:  
Si desarrollás software libre (FOSS) y querés combinarlo con OpenLaw, 
la versión open source de OpenLaw bajo AGPLv3 es la ideal.  
Si usás una licencia FOSS distinta a la AGPLv3, Fundación Clarius ofrece 
una excepción para las Bibliotecas Cliente de OpenLaw, permitiendo su 
uso sin que todo el trabajo derivado caiga bajo la AGPLv3, bajo ciertas 
condiciones. 

Ver [AGPLv3](https://opensource.org/license/agpl-v3) y 
[Excepción Universal FOSS](https://oss.oracle.com/licenses/universal-foss-exception/).
